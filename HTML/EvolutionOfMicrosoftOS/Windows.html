Windows Before the 2000's

A Brief Overview

Windows1

Windows1 is where it all started for the Windows Operating System. It was realesed in November og 1985, being the first ever graphical interface that Microsoft worked on. The one who lead the development was Bill Gates himself, the founder of Microsoft, runing on the MS-DOS.

Windows2

Windows2 was released two years after the Windows1 in December of 1987 with the ability of overlaping other Windows and introducing the minimize or maximize windows function insted of the "iconising" or "zooming". Windows2 also introduced the control panel with various system setting and configurartions which survies until today.

Windows3

Windows3 was the first Windows that required a hard drive luncher and was released in 1990. It saw a much more success and became a challenger to Apple's Macintosh and the Commodore Amiga graphical interfaces. It introduced the ability of runing MS-DOS programmes, bringing multitasking to original aplications and with 256 colours support that made for a more modern style of interface.

Windows3.1

Windows3.1 released in 1992, introducing TrueType fonts, which made Windows for a viable publishing platform for the first time.Unlike the previous version of Windows, Windows3.1 required 1MB of RAM to operate and it had MS-DOS support for programs that were controlled with a mouse. It also was the first Windows operating system to to be distributed on a CD-ROM.

Windows95

Windows95 was released in August of 1995, bringing the first ever Start button and Start menu. Also, it impemented the conscept of "plug and play", with the idea of conecting a peripheral so the operating system finds the appropriate drivers for it and make it work. Thought it did not always worked. It also brought the the 32-bit enviroment, the ask bar and the focused multitasking. Along with Windows95, came the Internet Explorel, though it required the user to have the Windows95 Plus. In later versions, the added the IE as a default aplication.

Windows98

Windows98 was released in June of 1998, being build on Windows95 and with it bringing IE4, Outlook Express, the Windows Addres Book, Microsoft Chat and the NetShow Player. It also added the back and forward buttons on the addres bar of Internet Explorer, along with other things. One of the bigest additions was introduced the Windows Driver Model for components and accessories, one drive that supported all the future version of Windows to come.

WindowsME

The WindowsME or Windows Milennium Edition was released in September of 2000, and it was aimed at consumers, while the Windows2000 had an aim for the enterprise market. Although many consider it a low point int the Windows series, WindowsME was the last Windows to be based on the MS-DOS, and the last in the Widnows9X line. it brought the IE 5.5, Windows Media Player 7, and Windows Movie Maker. Also Introducing the autocompleat function in the Winodws Explorer, but the operating system had manu bugs, failed to install properly in some instances and being over all poor.

Windows 2000

Windows 2000 was released along side with WindowsME, aimed at the enterprises, it was released in February of 2000, being based on Microsofts business-orientated system, which was the Windows NT. It was also called Windows 2k, with its code name during the development stage being Janus. It contains over 29 million lines of code, and it is mostly writen in the C++ programming language. There were four versions of Windows 2000 that came out, for the perpose of industrial use, those are the Professional, Server, Advanced Server and Datacenter Server, with limited edition packages. It had system support for the Windows NT file system,as well as other systems. It can be installed two ways, manualy or or with unattended installation, which relies on answer files to fill in information about the intsallation and can be made with the System Preparation Tool.

History Of Windows 2000

The Windows 2000 is the succesor to the Microsoft Windows NT family of operating systems, with Microsoft anouncing on Octomber 27 of 1998 that the final version of the Beta 2 NT 5.0 system, was going to be the Windows 2000, which the name was refering to the released date. It was first planed for Windows 2000 to replace both Winodws 98 and Windows 4.0, with that not becoming a reality. Instead, they made an update on Windows 98, and later an update on Windows 4.9, becoming Windows 4.9. With Windows 2000, the Start Menu saw the so called personalized manus, expandable special folders, and the ability to have multiple programs open at the same time.

Windows XP

Windows XP is considered one of the best Windows versions and it was released in Octomber of 2001. like Windows 2000 it was based in the Windows NT but with the consumer friendly elements of Windows 2000. Visualy diferent, with overhaul upgrades in the Start menu and task bar, having the familiar to all back-ground and the iconinc blue task bar and green Start Button. Windows XP was the longest running Microsoft OS, having three major updates and being supported until April of 2014. Though it had problems with security, even if it had a fire wall. Its success was also one of it's weakneses, as it cought the atention of many hackers and criminals, who in their part exploited its flaws, and especialy on the Internet Explorer.

Windows XP was active for almost six years before being replaced by Windows Vista, which updated the entire look and feel of Windows with focusing more on the transperent elements and it's security setings. But, this version of Windows did not make the same imperssion that Windows XP or the others did to the public. Windows Vista was buggy, spamed the user with many requests for app permissions under the User Acount Control, which made users to approve or not attemps for programs to make changes to the machine. It also had a problem running smoothly on older computers, even if it had the Vista Ready lael. Windows 7

Windows 7 for many was the true Windows update, or what should have been Windows Vista after Windows XP, making its apearence in Coctomber of 2009, fixing all the problems that Windows Vista had with come changes on the over all look of the start screen. Being faster and more reliable and easier to use, making it the OS taht users and companies chose to upgrade from Windows XP that had bypassed the Windows Vista uprade. It allowed for aplications to be pined to the start screen task bar for easier and faster access to programmes. It also added new security system, haveving a complete overview of it. Also some changes were made in the User Account Control system. Windows 8 And Windows 8.1

Windows 8 was released in Octomber of 2012 and it was a massive change for the Windows OS, as it created a whole new interface, without the Start button and menu being pressent. Insteat, it had all the nessesary programmes showing on the screen in order for it to be touch-friendly. It also was faster that any of the previous Windows and it had a USB 3.0 driver support, plus a tab for the new at the time Windows Shop. Thought user were still able to download thirf party programmes, they were not displayed in the main screen. But this new Windows Interface style was not that popular amongs the comunity, as they wanted to have the traditional way of navigation with the mouse and keyboard, but Microsoft wanted to balance it between touch support and desctop users, for many to conisder it a backstep fro the previous version of Windows. It also did not help that there not many touchscreens in use, so it made the whole touch mechanic unecessary.

Windows 8.1 which was an update from Windows 8, being made in Octomber of 2013, not only it brought back the Start Button along side with the Start Screen, it also allowed users to boot directly to the desktop of Windows 8.1, making it more comfortable to mavigate with a mouse and a kewboard.

Windows 10

Windows 10 is the latest and curent version of the Windows Operating System released to the public on July 29 of 2015. It released as a part of the Windows NT family and was the successor to Windos 8.1 that had released two year earlier. Windows 10 had a positive effect in the Windows series at the time of its release. It went back to the desctop interface just like the previous Windows as well the integration of the Xbox Live and the Cortana personal assistant, along with Microsoft Wolrd, which replaced the Internet Explorer.

Development Of Windows 10

In December of 2013, Microsoft was working on a new update for the Windows 8, with the name Threshold. They called Threshold a "wave of operating systems" troughout multiple platforms and it was ment to arive at 2015. Their aim was to unite the platform and toolkits for Windows and Xbox One. In April of 2014, they revieled a new version of the Windows 8.1 which inculded a more traditional way of navigating around, replacing the Windows 8 style, going back to the Windows 7 style of interface and start menu. On September of 2014, Microsoft anounced their new version of Windows, the Windows 10, calling it Microsoft's "most comprehensive platform ever", having a unified platform for desktop, laptops, tablets and even smartphones. Windows 10 made its release on June 29 of 2015 with advertising campaings begining on the 20 of July of the same year.
