The birth of the Microsoft Disk Operating System

MS-DOS, short for Microsoft Disk Operating System, is an operating system the x86-based computers, big part developed by Microsoft and its original writter was Tim Paterson and it was inroduced in August 1981, with it's last update released made in 1994. The operating system was originaly named QDOS, standing for Qucik and Dirty Operating System, before Microsoft purchased exlusive right to sell the system, ranaming it MD-DOS. Starting with Windows NT, the operating systems that Microsoft created were desing independently of MS-DOS, but were capable of runing some MS-DOS applications.

Although the system lacked multitasking, multuser capabilities and MS-DOS was limited to a command, it enjoyed enormous popularity in the 1980s nad early 1990s. It was limited to a command line interface in contrast to the more user friendly GUI. Even after MS-DOS ceased to be marketed as a stand-alone OS, it is still used in some computed systems today.
